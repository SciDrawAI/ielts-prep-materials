# 雅思学术类备考资料包 · IELTS Academic Prep Materials

一套自用整理的雅思学术类(IELTS Academic)备考材料,包含完整备考手册 PDF 和两个带字幕的音频练习视频。

## 内容

| 文件 | 说明 |
|------|------|
| `IELTS_Handbook.pdf` | 备考总手册(43 页):听力 / 阅读 / 写作 / 口语四科,每科按「题型介绍 → 技巧方法 → 语言材料 → 行动清单」编排,含高频词表、同义替换库、写作功能句骨架、口语母故事模板等 |
| `IELTS_Listening_Practice.mp4` | 听力练习视频(37 分钟):165 个高频答案词逐字母拼读 + 50 组同义替换 + 16 题听写靶场(含动笔静音时间,字幕不剧透答案)+ 信号词与场景套路 |
| `IELTS_Speaking_Practice.mp4` | 口语练习视频(12 分钟):Part 1 必考三话题问答 + Part 2 六个母故事 + Part 3 句型库,考官/考生双声音,逐句同步字幕 |

## 使用建议

- **听力视频**:通勤时循环听;听写部分请准备纸笔,每题后留有 4 秒作答时间
- **口语视频**:先听熟,再跟读模仿,最后脱稿复述(与原文有出入是好事,防背诵痕迹)
- **手册**:阅读与写作两科以纸面学习为主,配合手册中的行动清单推进

音频由 TTS 生成,发音供跟读参考。
